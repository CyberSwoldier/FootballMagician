# ⚽ Football Betting Probability Engine (Streamlit)

Professional football probability & bet-set engine using real data from SportMonks.

## Features
- Coverage of Europe's top leagues + UEFA competitions
- Poisson/xG-based probabilities
- Market-blended model
- 5+ diversified bet sets ≥70%
- Kelly staking
- Streamlit-ready

## Setup
1. Upload repo to GitHub
2. Add API key in Streamlit Secrets:
```
SPORTMONKS_API_KEY = "YOUR_KEY"
```
3. Run `streamlit run app/main.py`
